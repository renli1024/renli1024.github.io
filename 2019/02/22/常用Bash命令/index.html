常用Bash命令

Bash命令适用于Linux和Mac下的terminal
mkdir xxx：创建空目录（在当前目录下）
cd <目录路径>：转移到新目录。<目录路径>可以是绝对路径，也可以是当前目录的相对路径。
pwd：显示当前目录
ls：查看文件夹包含哪些文件
